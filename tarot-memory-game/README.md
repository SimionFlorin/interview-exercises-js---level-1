# Tarot Memory Game

A memory matching game built with React featuring tarot card pairs. Test your memory by flipping cards to find matching pairs!

## Features

- 12 cards (6 matching pairs) with tarot-themed symbols
- Card flip animations with 3D effects
- Move counter to track your performance
- Win detection with celebration message
- Reset functionality to start a new game
- Beautiful gradient UI with responsive design

## Game Rules

1. Click on a card to flip it face-up
2. Click on a second card to reveal it
3. If the cards match, they stay face-up
4. If the cards don't match, they flip back after 1 second
5. Continue until all pairs are found
6. Try to complete the game in as few moves as possible!

## Running the App

```bash
npm start
```

Runs the app in development mode.
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
